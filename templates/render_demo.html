✓这段内容是后端渲染出来的

你好，{{ name }}！现在是 {{ now }}， 系统读到你的心情是：{{ mood }}。

这一整段文字（包括名字、心情、时间）都不是写死在 HTML 里的， 而是 main.py 里的 render_demo() 函数 读取 URL 参数 ?name=...&mood=... 后，通过 Jinja2Templates().TemplateResponse() 塞进这个模板文件的 {% raw %}{{ name }}{% endraw %} / {% raw %}{{ mood }}{% endraw %} 占位符渲染出来的。

试试修改地址栏参数，回车看看效果：

  • /render_demo?name=小明&mood=兴奋
  • /render_demo?name=全老师&mood=淡定
  • /render_demo?name=六年级同学&mood=好奇
